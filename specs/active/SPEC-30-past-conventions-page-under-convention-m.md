  ## Summary

  Add a "Past Conventions" submenu item under the Convention menu. The page
  lets any site visitor (public, no login required) browse the history of OSA
  annual conventions by year. Content is authored and maintained in Sanity CMS.
  
  ## Default Page Behavior
  
  - Route: `/conventions/past` → redirects to
  `/conventions/past/[most-recent-year]`
  - Each year lives at `/conventions/past/[year]` (e.g.
  `/conventions/past/2025`)
  - A year-selector dropdown (most recent first) lets visitors navigate between
  years
  - Empty/unpublished convention years show an empty page — no placeholder
  needed

  ## Convention Header (per year)
  
  Title composed from: convention number (manual entry), year, city, state.
  Example: "OSA 56th Annual Convention 2025 at Dallas, TX"

  Additional optional header fields:
  - Convention dates (e.g. "July 4–6, 2025")
  - Venue name
  - Convention theme / motto
  - Host chapter or region
  ## Page Sections (per year)
  
  ### Overview
  Free-form rich text describing the convention.
  
  ### Convention Core Team
  List of { name, role } pairs. Example: "General Secretary — John Doe"
  
  ### Convention Guests
  List of { name, role } pairs. Example: "Chief Guest — Jane Smith, Consul
  General". No bio needed.
  
  ### Convention Donors
  Tiered donor list. Each tier has a heading (free text set by content author,
  e.g. "Platinum Sponsors") and zero or more donor entries: { name,
  organization (optional) }. Tier names and count are NOT predefined — authors
  define them freely each year.
  
  ### Convention Award Winners
  List of { award name, recipient name } pairs. No categories needed.
  
  ### Media
  Two optional external links (clickable buttons, no embedding, no thumbnails):
  - YouTube video link
  - Photo album link
  
  ## Access Control
  Public — no login required.
  
  ## Content Authoring
  All content managed in Sanity CMS, following the same integration used for
  the Programs menu (SPEC-15).
  
  ## Navigation
  "Past Conventions" added as a submenu item under the existing Convention menu
  (8-menu tree from SPEC-15). 
  
  ## Non-Goals
  - No search or filtering within a convention page
  - No member-gating
  - No file uploads (external links only for media)
  - No auto-calculation of convention number from a start year
